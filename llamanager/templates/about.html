{# ---- Project ---- #}
Project

llamanager is a single-host manager and queueing proxy for llama-server. It wraps a local inference engine so a single GPU can serve multiple clients through one OpenAI-compatible endpoint, with per-origin priority queuing, hot model swaps, crash supervision, and a web UI.
