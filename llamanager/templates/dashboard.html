{# ---- Quick reference (static, outside the polled div) ---- #}
Quick reference copy-paste examples
{# 1 · Chat completion #}
Chat completion 
curl {{ base_url }}/v1/chat/completions \
  -H "Authorization: Bearer $KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "any",
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'
{# 2 · Streaming #}
Streaming 
curl -N {{ base_url }}/v1/chat/completions \
  -H "Authorization: Bearer $KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "any",
    "stream": true,
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'
{# 3 · Text completion #}
Text completion 
curl {{ base_url }}/v1/completions \
  -H "Authorization: Bearer $KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "any",
    "prompt": "The meaning of life is"
  }'
{# 4 · List models #}
List models 
curl {{ base_url }}/v1/models \
  -H "Authorization: Bearer $KEY"
{# 5 · Request a specific model #}
Specific model / profile 
curl {{ base_url }}/v1/chat/completions \
  -H "Authorization: Bearer $KEY" \
  -H "X-Llamanager-Model: profile:my-profile" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "any",
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'
{# 6 · Python (OpenAI SDK) #}
Python · OpenAI SDK 
from openai import OpenAI

client = OpenAI(
    base_url="{{ base_url }}/v1",
    api_key="$KEY",
)

r = client.chat.completions.create(
    model="any",
    messages=[
        {"role": "user",
         "content": "Hello!"}
    ],
)
print(r.choices[0].message.content)
{# Footnote #}
Replace $KEY with your origin's API key from the Origins page.
The "model" field in the request body is required by the OpenAI format but llamanager ignores it. Use X-Llamanager-Model to select a specific model or profile.
