Slots

Run multiple LLM models in parallel, each in its own slot on its own port. Requests route by model id — no swap penalty when the requested model is already warm in some slot.

Slots is on. A few rules to keep in mind:
  • Exclusive mode is automatically disabled (the two are mutually exclusive — see the Launch page).
  • Requests that name an unloaded model are rejected with 409 — you own slot assignments, no auto-eviction.
  • VRAM is the operator's call. Loading a 32 GB-worth of models on a 32 GB card will OOM somewhere.
