{# ---- Section 4: Request-level overrides ---- #}
Model overrides for API clients

API clients can request a specific model or profile on a per-request basis by sending the X-Llamanager-Model header. If the requested model differs from what is currently loaded, llamanager will hot-swap automatically.

Load by profile name
X-Llamanager-Model: profile:qwen35-4b-default
Load by model ID
X-Llamanager-Model: unsloth/Qwen3.5-4B-GGUF/Q4_K_M.gguf

Each origin's allowed_models setting restricts which models it can request. Set to * on the Origins page to allow any model.
